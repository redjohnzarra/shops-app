Shops App using React Native
============================

How to use:
-----------

Clone this project
------------------
```
git clone https://github.com/redjohnzarra/shops-app.git
```
Go to the cloned project
------------------------
```
cd shops-app
```
Install node modules
--------------------
```
npm install or yarn install
```
Run the project
---------------
```
npm start or yarn start
```
